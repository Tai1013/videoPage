# Vue 3 + Vite

## [線上演示連結](https://tai1013.github.io/videoPage/)

---

### 專案啟動 [官方連結](https://vitejs.dev/)

```bash
#Vite 需要Node.js版本 >=12.2.0
$ npm create vite@latest
$ yarn create vite

#安裝本專案Node
$ npm install

#啟動專案
$ npm run dev
```
